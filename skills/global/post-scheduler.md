# Post Scheduling Skill

## Default Schedule
- Weekdays: 10:00 AM client timezone
- Never schedule past midnight
- Minimum 4 hours between posts on same platform

## Platform Rules
- Facebook: image + full caption
- Instagram: image + caption (max 2200 chars) + hashtags
- Reels: video + short hook caption

## Caption Format
- Lead with hook (first line most important)
- Use line breaks for readability
- CTA at end always

## Scheduling Logic
1. Determine client timezone from context
2. Calculate next available weekday slots
3. Alternate platforms when scheduling multiple posts
4. Confirm schedule before publishing
